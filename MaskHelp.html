Slit Mask Help

Slitmask Options:

Help for the inventory options:
  • MASK STATUS
    • UNMILLED

      A mask design that is UNMILLED means that either a mask has not been created for the mask, or the mask created has expired, been archived, and the design has been set to be re-milled. A design with status = UNMILLED will show up in the milling queue. The inital status of a mask upon submission is UNMILLED.

    • READY

      A mask design that is READY means that a mask has been created / milled. A READY mask is on Maunakea summit and ready to observe with, thus is either in the instrument or could be configured to be inserted into the instrument.

    • ARCHIVED

      An ARCHIVED mask design means the design will not be used to create a mask, if a mask has been created that mask is now archived (recycled) and no longer available for use. To observe with an ARCHIVED mask design, the mask must be set to 'RE-MILL' by a Keck Support Astronomer, which will put the design back into the milling queue.

  • Archive

    • The ARCHIVE button will mark masks as ARCHIVED. The meaning of archived is that the mask is no longer READY for observing. If it was milled, the physical metal mask is no longer available for use. To move a mask design from ARCHIVED to READY, the PI will need to contact someone at Keck Observatory to REMILL the mask.


  • Auto Archiving

    • Mask designs are automatically archived if they are 6 months past their use date. Once the mask design is marked as archived, the mask will be removed from Maunakea summit and no longer be available for use. An email is sent to the PI to 2 weeks before the mask is archved warn them of that this process, at this point they have the ability to update the use-date saving the mask from the recycle pile.


  • Remill

    • The remill functionality is only available to Keck Slitmask Admins. Remill allows a mask design to be sent to the milling queue. If an archived mask design needs to be used again, a new mask can be milled from the design by sending marking it to be remilled. Contact an Support Astronomer or coversheet@keck.hawaii.edu to have a mask remilled.


  • Plot

    • The Plot button produces a graphical representation of the mask design.


  • FITS file

    • The FITS file generates a 'FITS chunk' for any mask design. This can take a while to generate and is not useful unless you already know what it is.


  • Detail

    • The Detail button shows all the data contained in the mask design and blueprint(s).
